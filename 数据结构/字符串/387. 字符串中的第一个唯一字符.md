# 387. 字符串中的第一个唯一字符
给定一个字符串，找到它的第一个不重复的字符，并返回它的索引。如果不存在，则返回 -1。

示例：

    s = "leetcode"
    返回 0
    
    s = "loveleetcode"
    返回 2

 - 提示：你可以假定该字符串只包含小写字母。

解答

    /**
     * @param {string} s
     * @return {number}
     */
    var firstUniqChar = function(s) {
        let map = new Map()
        for (let i = 0; i < s.length; i++) {
            if (map.has(s[i])) {
                map.set(s[i], map.get(s[i]) + 1)
            } else {
                map.set(s[i], 1)
            }
        }
        for (let [key, value] of map.entries()) {
            if (value === 1) {
                return s.indexOf(key)
            }
        }
        return -1
    };
